# Install Intake-Thredds

Intake-Thredds can be installed in three ways:

```{eval-rst}

.. tab-set::

    .. tab-item:: pip

        Using the `pip <https://pypi.org/project/pip/>`__ package manager:

        .. code:: bash

            $ python -m pip install intake-thredds

    .. tab-item:: conda

        Using the `conda <https://conda.io/>`__ package manager that comes with the
        Anaconda/Miniconda distribution:

        .. code:: bash

            $ conda install intake-thredds --channel conda-forge

    .. tab-item:: Development version

        To install a development version from source:

        .. code:: bash

            $ git clone https://github.com/ncar-xdev/intake-thredds
            $ cd intake-thredds
            $ python -m pip install -e .

```
